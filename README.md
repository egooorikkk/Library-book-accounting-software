During the development of this program, various functions were created for working with book data. Each function performs a specific task, such as adding a book, deleting a book, editing book data, viewing all data, searching for a book by title, sorting a list of books by various criteria, and others. To improve the user experience, the ability to continue working after each operation has been implemented. The user can choose whether he wants to continue working with the program or terminate it. In general, this program provides a basic set of functions for managing a list of books.
